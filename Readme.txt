Клонирование проекта в папку nameProject:
git clone https://... nameProject

Добавить файлы в коммит:
git add.

аписать коммит:
git commit -m "Комментарий коммита"

Отправить коммит:
git push

Принять изменения:
git pull

Авторство коммитов:
git config --global user.emaill "Ваш емайл"
git config --global user.name "Ваше имя"
